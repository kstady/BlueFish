Keep-Change-Change Rule

The Keep-Change-Change rule is a clever way to remember how to rewrite any subtraction problem as an addition problem.

Keep-Change-Change Rule

Any subtraction problem can be rewritten as an equivalent addition problem using the steps below.

  1. Keep the sign of the first number.
  2. Change the operation of subtraction to addition.
  3. Change the sign of the second number.

Remember:  Subtracting a number is the same as adding the opposite of that number.

Use the Keep-Change-Change rule to find the difference \(-60 - (-10).\)

The diagram below demonstrates how to rewrite the subtraction problem \(-60 - (-10)\) as an addition problem. Starting from the left, we keep the sign of the first number, change the operation of addition to subtraction, then change the sign of the last number. Lastly, we add to find the result.

\begin{eqnarray*} -60 - (-10) &=& -60 + 10 \\ &=& -50 \end{eqnarray*}

Use the Keep-Change-Change rule to find the difference \(40-55.\)
